# RSS READER

## Quickstart

### Client

  1) Install:
  ```
  # cd rss_client
  # npm install
  ```
  2) Run dev server:
  ```
  # npm start
  ```

### Server

  1) Install:
  ```
  # cs rss_server
  # pip install -r requirements.txt
  ```
  2) Run migrations
  ```
  # python manage.py migrate
  ```
  3) Run xss parser
  ```
  # python manage.py parserssfeeds
  ```
  4) Run dev server
  ```
  # python manage.py runserver
  ```
